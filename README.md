# Starlit - Library Website

## Overview

Starlit is a responsive library website developed as a university project. The website was designed to provide users with an interactive platform to explore books and library-related content across multiple devices. The project focuses on creating a user-friendly interface while ensuring a consistent experience on desktop, tablet, and mobile screens.

## Features

* Responsive design for multiple screen sizes
* Modern and intuitive user interface
* Book browsing and search functionality
* Interactive navigation and content display
* Mobile-friendly layout
